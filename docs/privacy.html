3. Type and Scope of Data Processing

Our activities include research, data analysis, data engineering, and data mining. We generally do not process personal data as defined in Art. 4(1) GDPR. If, in exceptional cases, personal data is provided to us in the context of projects, it is anonymized before any processing, preventing any link to identifiable individuals. We do not share data with third parties, nor do we analyze personal data using external services.

Task Manager App "Procrastino"

Our mobile app enables synchronization of tasks with Google Tasks. Signing in with a Google account is required for this purpose.
Processed data: Only the information required for synchronization (e.g., task contents, task metadata) is processed.
Storage: All data is stored exclusively on your device in the app’s storage. No storage or processing takes place on servers operated by us.
Disclosure: No data is shared with third parties.
Advertising/Tracking: The app does not contain any advertisements and does not use analytics tools.
Google services: The use of Google Tasks is additionally subject to Google’s Privacy Policy: https://policies.google.com/privacy

4. Hosting via GitHub Pages

Our website is hosted via GitHub Pages (GitHub Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA; a company of Microsoft Corporation). When accessing the website, GitHub automatically processes technical information (e.g., IP address, browser type, operating system, referrer URL, timestamp) in server log files to ensure operation and security. For further details, please refer to GitHub’s Privacy Statement: GitHub Privacy Statement. We do not analyze these log files ourselves.

5. Legal Basis

Since we do not process personal data ourselves, our activities are not based on consent or legitimate interest. The logging carried out by GitHub is based on Art. 6(1)(f) GDPR (legitimate interest in operation, security, and stability of web services) by GitHub.
